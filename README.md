# simple-webpack-develop-config
===============================
simple webpack develop config file

### prepare list
`
    > npm install webpack webpack-dev-server -g
`
<br/>
`
    > npm install -g typescript typings webpack
`
<br/>
`
    > npm install
`
<br/>
`
    > npm link typescript
`

##### entry folder
    > dist/
##### entry root
    > index.html

### Star to begint
`
    > npm run dev
`
<br/>
`
    > http://localhost:8080/webpack-dev-server/index.html
`
